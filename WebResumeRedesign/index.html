This is a web application that I developed to help track my own progress in the gym better. It features several input forms where the user can submit information about how much weight they lifted and on which days. It then calculates and displays helpful charts and statistics to help the user quantify their progess. I made this application because it is something that I am passionate about and because I wanted to improve my own web development skills.
